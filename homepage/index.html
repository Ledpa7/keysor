Unrivaled Precision Control

Powered by pure Rust without a garbage collector and Windows native low-level APIs, breaking through system resource limits.

0ms

Low-level OS Input Hooking

Intercepts low-level keyboard hooks (`WH_KEYBOARD_LL`) directly, achieving 0ms latency response. Deep integration with OS thread prevents freezing.

🧲

Magnet Snapping (Hysteresis)

Automatically snaps to the center of nearby interactive UI elements (buttons, icons, tabs, etc.) within 25px. Incorporates a 15px hysteresis algorithm for smooth detaching.

📐

Diagonal Speed Correction

Corrects diagonal movement speeds using a 1/sqrt(2) (≈ 0.707) trigonometric scaling factor, ensuring the cursor moves at a constant speed in all directions.

🔍

Sub-pixel Accumulation

Accumulates fractional distance remainders across frames, ensuring ultra-smooth micro-movements and seamless linear acceleration.

⚡

High-Precision 100Hz Thread

Runs calculations on a dedicated 100Hz background thread mapped with DPI scales, maintaining a lag-free control loop even under heavy system loads.

🔋

Ultra-lightweight Resource Usage

Runs as a single portable binary of under 3MB and consumes less than 2MB of memory. Limits CPU and GPU usage to under 0.1% for maximum efficiency.

Keysor Hotkeys & Control Layers

Manage three ergonomic control directions and helper shortcut sequences simultaneously with a single Caps Lock modifier key.
